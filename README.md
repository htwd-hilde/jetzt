# HILDE Website

Eine Website für das HILDE-Projekt: Haptische Interfaces für lebendige, digitale Erfahrungen.

## Installation auf GitHub Pages

### Schritt 1: Repository erstellen
1. Gehen Sie zu GitHub und erstellen Sie ein neues Repository
2. Benennen Sie es z.B. `hilde-website`
3. Setzen Sie es auf "Public"

### Schritt 2: Dateien hochladen
1. Laden Sie folgende Dateien in Ihr Repository hoch:
   - `index.html`
   - `styles.css`
   - `README.md`

### Schritt 3: GitHub Pages aktivieren
1. Gehen Sie zu den Repository-Einstellungen (Settings)
2. Scrollen Sie zu "Pages" im linken Menü
3. Wählen Sie unter "Source": Branch `main` (oder `master`)
4. Wählen Sie als Ordner: `/ (root)`
5. Klicken Sie auf "Save"

### Schritt 4: Website aufrufen
Nach wenigen Minuten ist Ihre Website unter folgender URL erreichbar:
```
https://[ihr-benutzername].github.io/hilde-website/
```

## Struktur

- `index.html` - Hauptdatei mit allen Inhalten
- `styles.css` - Styling und Layout
- `README.md` - Diese Anleitung

## Anpassungen

### Farben ändern
Die Hauptfarbe (Blau) kann in der `styles.css` geändert werden:
- Primärfarbe: `#4169E1`
- Sekundärfarbe: `#5B7FE8`

### Inhalte bearbeiten
Alle Texte können direkt in der `index.html` bearbeitet werden.
